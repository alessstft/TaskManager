### Task Manager

**Обзор проекта**  
Диспетчер задач — это приложение для мониторинга системных ресурсов в операционной системе Windows. С помощью этого инструмента можно отслеживать текущие процессы, службы, а также производительность системы. Программа предоставляет графический интерфейс для удобной работы с данными, а также позволяет завершать процессы и получать детализированную информацию о системе.

**Основные функции**

😃Мониторинг процессов

Вкладка "Процессы" отображает список всех активных процессов на вашем компьютере.

Вы можете завершить процесс, выбрав его и нажав кнопку "Завершить процесс". Это может быть полезно, если какой-то процесс не отвечает или сильно загружает систему.

☝Мониторинг служб

Вкладка "Службы" позволяет увидеть состояние всех системных служб.

В этой вкладке вы также можете остановить или перезапустить службы, если это необходимо для оптимизации работы системы.

☝Мониторинг производительности

Вкладка "Производительность" отображает текущие показатели работы системы, такие как использование CPU, RAM, диска и сети.

Нажав на любую из кнопок в этой вкладке, вы получите более детализированную информацию о состоянии этих ресурсов.

☝Завершение процесса

Для завершения процесса выберите его в списке на вкладке "Процессы" и нажмите кнопку "Завершить процесс". Это полезно, если какой-то процесс замедляет работу вашего компьютера или завис.
    
## Описание зависимостей

1. **Python 3.9+**
    - Основной язык разработки. Версия 3.9 или выше обеспечивает поддержку всех необходимых функций и библиотек.
    
2. **Windows Api 0.3**
    - Набор базовых функций интерфейсов программирование приложений операционных систем семейств Microsoft Windows. 
    - Предоставляет прямой способ взаимодействия приложений пользователя с операционной системой _Windows_.
    
3. **sysinfo 0.29**
    - Rонтейнер, используемый для получения информации о системе.

4. **Serde 1.9:**
    - Фреймворк для эффективной и универсальной сериализации и десериализации структур данных Rust.

**5. serde_json 1.0**  
- Высокопроизводительная библиотека для работы с JSON в Rust.  
- Обеспечивает парсинг и генерацию JSON-данных.  
- Используется для сериализации и десериализации данных о процессах и системной информации.

**6. lazy_static 1.4**  
- Макрос для Rust, позволяющий инициализировать статические переменные с ленивой инициализацией.  

**7. libc 0.2**  
- Rust-биндинги к библиотеке libc, предоставляющие низкоуровневый интерфейс к API операционной системы.  

**8. num_cpus 1.15**  
- Простая библиотека для определения количества CPU в системе.  
- Предоставляет информацию о количестве физических и логических процессоров.  

**9. ctypes**  
- Стандартная библиотека Python для работы с C-совместимыми типами данных.  
- Позволяет вызывать функции из динамических библиотек и взаимодействовать с кодом на C.  

**10. tkinter**  
- Стандартная графическая библиотека Python для создания кроссплатформенных GUI-приложений.  
- Используется как основа графического интерфейса Task Manager.
